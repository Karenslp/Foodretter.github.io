Foodretter: Análisis del Desperdicio Alimentario

Descripción del Proyecto

Este proyecto tiene como objetivo analizar el desperdicio de alimentos a nivel mundial, nacional (Colombia) y local (Bucaramanga), utilizando datos en formato CSV. Los resultados se presentan mediante gráficos generados con Python, Pandas, Matplotlib y Streamlit.

Herramientas Utilizadas

  • Pandas: para cargar y organizar los datos
  • Matplotlib: para crear los gráficos de barras
  • Streamlit: para visualizar los datos como una página web

Resultados Visuales

Funcionamiento General

El programa carga un archivo CSV con información de toneladas de comida desperdiciada. Luego, genera automáticamente gráficos diferenciados por colores según el lugar y nivel, usando funciones en Python.
